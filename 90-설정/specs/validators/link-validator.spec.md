# 링크 검증 Spec

## 필수 검증 (에러)
```markdown
- [ ] 자료정리: source 필드 존재?
- [ ] 자료정리: source 파일 존재?
- [ ] 핵심개념: source 존재?
- [ ] 핵심개념: MOC 연결?
- [ ] MOC: 핵심개념 3개+?
- [ ] 완료 즉흥메모: 결과 링크?
```

## 권장 검증 (경고)
```markdown
- [ ] 핵심개념: 개념연결 2개+?
- [ ] 핵심개념: 백링크 1개+?
- [ ] 자료정리: 핵심개념 파생?
- [ ] MOC: 관련 MOC 연결?
```

## 품질 검증 (제안)
```markdown
- [ ] 링크 관계 명시?
- [ ] 양방향 링크?
- [ ] 태그 일관성?
```

## 에러 메시지 형식
```
❌ 필수 링크 누락
핵심개념은 MOC에 등록되어야 합니다.

제안:
1. [[맵-AI시스템]]에 추가
2. 새 MOC 생성
```

## 경고 메시지 형식
```
⚠️ 개선 제안
개념 연결이 2개 미만입니다.
관련 개념: [[개념A]], [[개념B]]
```

## 자동 수정 가능 항목
- source 필드 추가
- MOC 역링크 추가
- 태그 정규화
- 파일명 정규화
