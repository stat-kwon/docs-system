# 구조 Spec

## 폴더 계층구조
```
docs-system/
├── 10-수집/               # 정보 수집
│   ├── 원문/              # 원본 자료 (수정 금지)
│   └── 즉흥메모/           # 빠른 캡처
├── 20-정리/               # 정보 가공
│   ├── 자료정리/           # Literature Notes
│   └── 핵심개념/           # Permanent Notes
├── 30-연결/               # MOC
├── 40-실행/               # 프로젝트
├── 80-보관/               # 보관함
│   └── 첨부파일/           # 원문의 첨부파일 저장
│       └── YYYYMMDD/      # 날짜별 폴더 구조
│           ├── 이미지.png
│           └── 문서.pdf
├── 90-설정/               # 설정/템플릿
└── 99-Todo.md            # 자동 집계
```

## 파일명 규칙

| 노트 타입 | 형식 | 예시 |
|----------|------|------|
| 원문 | `YYYYMMDD-출처.md` | `20241024-claude.md` |
| 즉흥메모 | `YYYYMMDD-HHMM-설명.md` | `20241024-1430-아이디어.md` |
| 자료정리 | `정리-YYYYMMDD-주제.md` | `정리-20241024-AI.md` |
| 핵심개념 | `개념-YYYYMMDDa-이름.md` | `개념-20241024a-에이전트.md` |
| MOC | `맵-주제.md` | `맵-AI시스템.md` |

## 기본 규칙
- 모든 문서는 `.md` 형식
- Wikilink 사용: `[[노트명]]`
- 태그 계층구조: `#태그/하위태그`
- 원문은 절대 수정 금지

## 첨부파일 관리 규칙
- 원문의 모든 첨부파일은 `/80-보관/첨부파일/YYYYMMDD/` 하위 저장
- 날짜 폴더명은 원문 파일명의 날짜와 일치
- 파일명 충돌 방지: `파일명-1.확장자` 형식으로 순번 추가
- 지원 형식: 이미지(png, jpg, gif), 문서(pdf, docx), 다이어그램(drawio)
