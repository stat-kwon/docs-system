# 프로젝트 시나리오 Spec

## 트리거 키워드
- "프로젝트 시작", "새 프로젝트"
- "실행 계획", "프로젝트 폴더"

## 프로젝트 생성 규칙

### 1. 사용자 확인 필수
```
⚠️ "프로젝트 폴더를 생성하시겠습니까?"
```

### 2. 템플릿 사용
```
Filesystem:read_file("/90-설정/템플릿-Project.md")
```

### 3. 폴더 구조
```
/40-실행/프로젝트명/
├── 프로젝트명.md          # 프로젝트 개요 (메인 파일)
├── 프로젝트명-계획.md      # 계획 문서
├── 프로젝트명-자료.md      # 참조 자료
└── 프로젝트명-작업.md      # 작업 목록
```

### 4. 필수 YAML 필드
```yaml
---
type: project
status: active|completed|archived
created: YYYY-MM-DD
tags: [project/category]
---
```

### 5. 초기 파일 생성
- `프로젝트명.md`: 프로젝트 개요, 목표, 범위 (메인 파일)
- `프로젝트명-계획.md`: 단계별 계획, 일정
- `프로젝트명-자료.md`: 관련 핵심개념/자료정리 링크
- `프로젝트명-작업.md`: Tasks 플러그인 형식 작업 목록

## 링크 규칙 (실행 연결)
- 프로젝트 → 핵심개념/자료정리 (자유)
- 역방향 링크 불필요
- 관련 MOC 참조 (선택)

## 작업 관리
```markdown
## Tasks
- [ ] 작업1 📅 2024-11-15 🔴
- [ ] 작업2 📅 2024-11-20 🟡
- [ ] 작업3 🟢
```
